# 生活服务接口

## index - 生活指数

15 种生活指数，覆盖穿衣、雨伞、感冒、洗车、旅游等。

```bash
moji-use-weather index <位置>
```

**示例**:
```bash
moji-use-weather index 北京
moji-use-weather index 成都
```

**15 种指数类型**:
| 编号 | 名称 | 用途 |
|------|------|------|
| 1 | 穿衣指数 | 穿什么衣服 |
| 2 | 紫外线指数 | 防晒需求 |
| 3 | 洗车指数 | 是否适合洗车 |
| 4 | 旅游指数 | 出游建议 |
| 5 | 感冒指数 | 感冒风险 |
| 6 | 运动指数 | 户外运动建议 |
| 7 | 雨伞指数 | 要不要带伞 |
| 8 | 舒适度指数 | 体感舒适度 |
| 9 | 晾晒指数 | 是否适合晾衣服 |
| 10 | 钓鱼指数 | 是否适合钓鱼 |
| 11 | 化妆指数 | 化妆建议 |
| 12 | 过敏指数 | 花粉/过敏风险 |
| 13 | 交通指数 | 路况影响 |
| 14 | 防晒指数 | 防晒等级 |
| 15 | 空调指数 | 是否需要开空调 |

**匿名试用**: 不支持

## aqi - 空气质量

实时 AQI 及主要污染物数据。

```bash
moji-use-weather aqi <位置>
```

**示例**:
```bash
moji-use-weather aqi 广州
```

**返回字段**:
- `aqi` - AQI 指数
- `level` - 等级 (优/良/轻度污染/中度/重度/严重)
- `pm25` / `pm10` - PM2.5 / PM10
- `so2` / `no2` / `co` / `o3` - 主要污染物
- `primaryPollutant` - 首要污染物

**匿名试用**: 不支持

## alert - 天气预警

当前生效的气象预警信息。

```bash
moji-use-weather alert <位置>
```

**示例**:
```bash
moji-use-weather alert 深圳
```

**返回字段**:
- `title` - 预警标题
- `type` - 预警类型 (暴雨/寒潮/大风/台风等)
- `level` - 预警等级 (蓝/黄/橙/红)
- `content` - 预警内容
- `pubTime` - 发布时间

**匿名试用**: 不支持
