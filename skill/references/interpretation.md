# 结果解读规则

当返回天气数据时，不要只返回原始 JSON。根据以下规则给出自然语言解读。

## 温度体感

| 温度范围 | 描述 | 穿着建议 |
|---------|------|---------|
| <= -10°C | 极寒 | 厚羽绒服+帽子围巾手套 |
| -10 ~ 0°C | 严寒 | 羽绒服+保暖内衣 |
| 0 ~ 5°C | 寒冷 | 厚外套或薄羽绒服 |
| 5 ~ 15°C | 偏凉 | 外套+长裤 |
| 15 ~ 25°C | 舒适 | 薄外套或长袖 |
| 25 ~ 30°C | 温热 | 短袖+薄裤 |
| 30 ~ 35°C | 炎热 | 透气短袖短裤 |
| >= 35°C | 酷热 | 尽量减少户外活动 |

**注意**: 优先使用体感温度 (realFeel) 而非气温。

## AQI 等级

| AQI 范围 | 等级 | 建议 |
|---------|------|------|
| 0-50 | 优 | 适宜户外活动 |
| 51-100 | 良 | 可正常活动 |
| 101-150 | 轻度污染 | 敏感人群减少户外，建议戴口罩 |
| 151-200 | 中度污染 | 减少户外活动 |
| 201-300 | 重度污染 | 避免户外活动 |
| > 300 | 严重污染 | 留在室内，紧闭门窗 |

## 降水概率

| 概率范围 | 描述 | 建议 |
|---------|------|------|
| 0-20% | 基本不会下雨 | 无需带伞 |
| 20-50% | 有可能下雨 | 建议带伞 |
| 50-80% | 较大概率下雨 | 务必带伞 |
| > 80% | 几乎肯定下雨 | 带伞+防水鞋 |

## 风力等级

| 风力 | 描述 | 影响 |
|------|------|------|
| 0-2 级 | 微风 | 无影响 |
| 3 级 | 轻风 | 树叶摇动 |
| 4 级 | 和风 | 灰尘飞扬 |
| 5 级 | 清风 | 小树摇摆，注意出行 |
| 6 级 | 强风 | 大风预警，注意安全 |
| >= 7 级 | 疾风+ | 避免户外活动 |

## 紫外线指数

| UVI 范围 | 等级 | 建议 |
|---------|------|------|
| 0-2 | 弱 | 无需防晒 |
| 3-5 | 中等 | 建议涂防晒霜 |
| 6-7 | 强 | 必须防晒，戴帽子墨镜 |
| 8-10 | 很强 | 避免正午外出 |
| >= 11 | 极强 | 尽量不要外出 |

## 预警等级

| 颜色 | 严重程度 | 行动建议 |
|------|---------|---------|
| 蓝色 | 一般 | 关注天气变化 |
| 黄色 | 较重 | 做好防范准备 |
| 橙色 | 严重 | 采取防护措施，减少外出 |
| 红色 | 特别严重 | 立即采取紧急措施，停止户外活动 |

## 输出格式要求

1. 先给出**一句话总结**（适合直接回复用户）
2. 再给关键数据细节（温度、天气、风力等）
3. 最后给出**行动建议**（穿什么/带什么/注意什么）

**示例**:
> 北京现在 28°C 多云，体感温度 30°C，比较热。建议穿短袖薄裤，外出注意防晒（紫外线中等）。空气质量良，适合户外活动。
